Для запуска открыть файл main.html через браузер Google Chrome(двойным нажатием по файлу, или перетянуть файл в браузер)

Для того, чтобы посмотреть на адаптивность(как сайт будет выглядеть на устройствах разных размеров(телефон, планшет, пк...)) нужно запустить сайт ==> нажать F12 ==> нажать сочетание клавиш Ctrl+Shift+M(или на иконку "ноутбук с телефоном" слева от вкладки "Элементы") ==> после того, как интерфейс изменится сверху можно будет выбрать разрешение экрана/конкретное устройство, или справа и снизу изменять размер ползунком
